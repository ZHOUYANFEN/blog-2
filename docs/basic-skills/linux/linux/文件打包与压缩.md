# 文件打包与压缩

## 常见压缩文件后缀

| 文件后缀 | 说明                          |
| -------- | ----------------------------- |
| .bz2     | 用 bizp2 压缩的文件           |
| .gz      | 用 gizp 压缩的文件            |
| .xz      | 用 xz 压缩的文件              |
| .tar     | 用 tar 打包的文件             |
| .tbz     | tar 打包时用 bizp2 压缩的文件 |
| .tgz     | tar 打包时用 gzip 压缩的文件  |
| .zip     | 用 zip/winzip 压缩的文件      |
| .rar     | 用 rar 压缩的文件             |
| .7z      | 用 7za 压缩的文件             |
|          |                               |
|          |                               |
