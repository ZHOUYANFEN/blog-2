# Python 常用魔术方法

介绍一些 Python 常用的魔术方法，如 ```__init__```、```__new__``` 等。

##  ```__init__```

这是一个实例函数，主要是用来构建对象属性的，当对象创建的时候，就会自动调用这个函数，如果有属性时，会给对象赋予相应的属性

```python
#__init__函数
class A:
    def __init__(self, name):  # 实例方法
        print('开始给对象赋予属性了')
        self.name = name
a = A("hello")
print(a.name)
--------------------------------------------
开始给对象赋予属性了
hello
```
